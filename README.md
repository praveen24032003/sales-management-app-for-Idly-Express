## 📖 Overview

**Idly Express** is a lightweight Flutter application designed to help small food‑service businesses track daily sales, profit margins, and inventory in real‑time. The app provides a clean dashboard, profit visualisations, and easy data export/import capabilities.

## ✨ Features

- **Real‑time dashboard** showing total sales, profit, and recent transactions.
- **Profit visualisation** with interactive charts (fl_chart).
- **Data export** to CSV/Excel and sharing via `share_plus`.
- **Mock data generator** for quick testing and UI prototyping.
- **State management** powered by `provider`.
- **Local persistence** using `sqflite`.

## 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/praveen24032003/sales-management-app-for-Idly-Express.git
   cd idly_express
   ```
2. **Install Flutter dependencies**
   ```bash
   flutter pub get
   ```
3. **Run the app**
   ```bash
   flutter run
   ```

> The project targets Flutter **3.10** (see `pubspec.yaml`).

## 🚀 Usage

- The app launches to the **Dashboard** where you can see a summary of sales.
- Navigate to **Profit** to view profit charts.
- Use the **Reports** screen to see a list of all sales entries.
- Tap the **Mock Data** button (top‑right) to generate random sales data for testing.
- Export data via the **Export** button in the header.
